# Progress Bar

This is a simple C++ progress bar you can use. It is available as a library and a c++ file.
Feel free to customise the code how you like!

# How to use:
```cpp
#include <iostream>
#icnlude "progressBar.hpp"

int main() {
  loadingBar(sleepTime) // sleepTime time between progress, 1000 = 1 Second.
}
```
