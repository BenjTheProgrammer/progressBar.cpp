# Progress Bar

This is a simple C++ progress bar you can use. It is available as a library and a c++ file.
Feel free to customise the code how you like!

# How to use:
```cpp
#include <iostream>

int main() {
  loadingBar(
  int finishpercent, // the amount of progress it needs to be finished.
  int secsleep // the amount of time between progress.
  ) 
}
```
